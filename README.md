# game2048
«2048» — это браузерная головоломка для одного игрока.

Основное:

игровое поле — сетка 4 × 4;

цель — создать плитку с числом 2048 (можно продолжать и после этого);

Как играть:

Игрок сдвигает все плитки на поле в одну из четырёх сторон (стрелками или свайпами).

Если две плитки с одинаковым числом «сталкиваются», они сливаются в одну с суммой значений (например, 2 + 2 = 4).

После каждого хода на свободном месте появляется новая плитка — 2 (с вероятностью 90 %) или 4 (10 %).

За каждое слияние начисляются очки — столько, каково значение новой плитки.

Когда игра заканчивается:

если на поле нет пустых клеток и невозможно сделать ход (нет соседних плиток с одинаковыми числами);

либо когда игрок достигает цели и решает остановиться.

Особенности:

максимальная возможная плитка на стандартном поле — 131 072;

стратегия часто строится на том, чтобы держать самую крупную плитку в углу и группировать большие числа вдоль края.
